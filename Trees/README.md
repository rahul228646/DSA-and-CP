# Notes

### Binary Tree :

    Searching  : O(n)
    Insertion  : O(n)
    Deletion   : O(n)

### Binary Search Tree :

    Searching  : O(n) (worst case) O(h) (general)
    Insertion  : O(n) (worst case) O(h) (general)
    Deletion   : O(n) (worst case) O(h) (general)

### AVL/ Height Balanced Tree :    

    Searching  : O(logn)
    Insertion  : O(logn) 
    Deletion   : O(logn)
